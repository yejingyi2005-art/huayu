> 基于 Figma Make 导出代码分析整理

## 一、设计语言

### 设计哲学
**手帐感 × 绘本感 × Calm Technology**

- **手帐感**：每一次记录都像贴上一张纸片
- **绘本感**：每个入口都像翻页而不是冲刺
- **Calm Technology**：每个反馈都用天气、季节和植物生长表达，让用户感到被欢迎，而不是被打扰

### 设计关键词
温暖 · 安静 · 自然 · 低刺激 · 手作感 · 生长感

## 六、间距与布局

### 网格系统
- `max-w-6xl` 内容区最大宽度
- 两栏网格：`md:grid-cols-[0.92fr_1.08fr]`（左 46% / 右 54%）
- 响应式断点：mobile 单栏 → `md:` 双栏

### 内边距
- 主容器：`px-5 py-8 md:px-10`
- 卡片内边距：`p-5 md:p-7`（主卡片）、`p-6`（功能卡片）、`p-4`（天气/记忆卡片）

### 间距
- 网格间距：`gap-10`（主）、`gap-5`（功能行）、`gap-4`（记忆卡片）、`gap-3`（天气）
- 垂直间距：`space-y-5`（右栏）、`mt-6`、`mt-8`、`mt-9`

## 八、动效系统

### 技术栈
- `motion`（Framer Motion v12 fork）

### 预设动画

| 元素 | 动画 | 时长 | 类型 |
|------|------|------|------|
| 云朵标签 | `y: [0, -8, 0], x: [0, 10, 0]` | 14s | 漂浮 |
| 左侧树 | `rotate: [-2, 2, -2]` | 5.5s | 摇摆 |
| 右侧树 | `rotate: [1, -2, 1]` | 6.5s | 摇摆 |
| 记忆卡片 | `opacity: 0→1, y: 16→0, rotate` | 0.7s + 0.12s stagger | 错开入场 |
| 嫩芽图标 | `y: [0, -10, 0]` | 8s | 浮动 |

### CSS 过渡
- CTA 按钮：`transition duration-500`
- 悬停旋转：`hover:-rotate-1` / `hover:rotate-1`
- 箭头位移：`group-hover:translate-x-1`

### Radix 状态动画
- `animate-in` / `animate-out`
- `fade-in-0` / `fade-out-0`
- `zoom-in-95` / `zoom-out-95`
- `slide-in-from-*`

## 九、组件设计

### 品牌装饰组件

#### PaperTape（纸胶带）
- 绝对定位、旋转 `-5deg` / `3deg`
- 尺寸：`h-7 w-24`，`rounded-sm`
- 半透明背景：`bg-[#F4E9C8]/70` / `bg-[#DDE6EC]/70`

#### DoodleIsland（手绘小岛）
- CSS 绘制的手绘风格小岛插图
- 有机圆形（`border-radius` 不对称百分比）
- 包含：云朵（动画）、2 棵树（动画）、树干、水池、"春天的票根"标签、"第 128 天"徽章、白色装饰点

│  │  │              │  │              │             ││
│  │  │  - 纸胶带     │  │  1. 共同时间  │             ││
│  │  │  - 品牌徽章   │  │  2. 低刺激    │             ││
│  │  │  - 标题      │  │  3. 情绪天气  │             ││
│  │  │  - 描述      │  │  4. 记忆卡片  │             ││
│  │  │  - CTA 按钮  │  │  5. 设计哲学  │             ││
│  │  │  - 小岛插图  │  │              │             ││

│  ┌── Sprout 浮动图标 (fixed bottom-6 right-6) ────┐  │
│  └──────────────────────────────────────────────────┘│

### P1 启动/品牌区（左栏 Hero）
1. **纸胶带装饰**：旋转 `-5deg`，位于左上角
2. **品牌标识**：Leaf 图标 + "花屿 Huayu"，圆角胶囊标签
3. **关系收藏馆**：粉色标签
4. **TODAY'S QUIET PAGE**：宽间距英文标签
5. **主标题**：Long Cang 手写体 "今天也在悄悄生长"，超大字号
6. **副标题**：产品定位描述
7. **CTA 按钮**：
   - 主要："写下一片记忆"（鼠尾绿填色，hover 旋转 + 箭头位移）
   - 次要："翻看旧相册"（奶油白描边，hover 旋转）
8. **DoodleIsland**：CSS 手绘小岛插图

### P2 功能卡片区（右栏 Features）
1. **共同时间卡片**：显示 "128 天"（Long Cang 字体），Wind 图标，小岛状态描述，虚线分隔线
2. **低刺激原则卡片**：三条原则卡片
   - 不催促打开
   - 不显示未读压力
   - 只提醒值得纪念的慢时刻
3. **情绪天气四宫格**：
   - 晴天 → 开心（`#F4E9C8`）
   - 微风 → 平静（`#DDE6EC`）
   - 阴天 → 疲惫（`#EFEDE4`）
   - 小雨 → 难过（`#F6E8EC`）
4. **记忆卡片**（3 张，交错入场动画）：
   - 日期 + 标题 + 笔记 + 天气标签
   - 纸胶带装饰 + 轻微旋转
5. **设计哲学横幅**：虚线边框，Feather 图标
   - "手帐感 × 绘本感 × Calm Technology"

### P3 浮动元素
- **Sprout 图标**：`fixed bottom-6 right-6`，浮动动画，半透明背景

## 十一、交互原则

### CTA 按钮交互
- 主要按钮 hover：`rotate(-1deg)` + 箭头右移
- 次要按钮 hover：`rotate(1deg)`
- 过渡：`duration-500`

### 低刺激原则（UI 层面）
- 无红点提醒
- 无未读状态
- 无排行榜
- 无等级体系
- 无打卡/签到
- 无点赞/评论

### 反馈方式
- 情绪 → 天气映射
- 时间 → 季节变化
- 记录 → 空间/植物生长
- 久别 → 重逢体验文案

## 十四、设计约束

### 色彩约束
- 避免使用饱和色
- 所有颜色保持低饱和度、暖色调
- 边框使用暖褐色而非灰色

### 交互约束
- 避免加载态反馈（无 skeleton 以外的 loading 状态）
- 避免弹窗打扰用户
- 避免未读标记和数字红点

### 内容约束
- 文案风格：诗意、温和、自然比喻
- 中英文混排时注意字体搭配（Long Cang 仅用于标题/装饰）
- 日期格式：`MM.DD`（如 "03.18"）

---

> **设计原则校验**：任何新增 UI 元素，必须回答：
> *"这个设计是在帮助用户收藏关系，还是在要求用户维护关系？"*
> 如果答案是后者，就不进入 MVP。
